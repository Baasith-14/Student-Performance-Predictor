# 🎓 Student Performance Predictor

## 📖 Overview

Student Performance Predictor is a Machine Learning web application that predicts a student's academic performance based on various input factors. The project is built using Python, Scikit-learn, and Streamlit to provide a simple and interactive prediction interface.

---

## 🚀 Features

- Predict student performance using Machine Learning
- Interactive Streamlit web interface
- Clean and user-friendly design
- Fast prediction results
- Easy to extend with new datasets

---

## 🛠 Tech Stack

- Python
- Scikit-learn
- Pandas
- Streamlit
- Joblib

---

## ⚙️ Installation

```bash
git clone https://github.com/Baasith-14/Student-Performance-Predictor.git

cd Student-Performance-Predictor

pip install -r requirements.txt

streamlit run app.py
```
